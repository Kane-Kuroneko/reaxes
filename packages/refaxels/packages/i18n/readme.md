# reaxel-i18n


## 为什么是refaxel而不是reaxel?
你的应用中也许有多层i18n的需求,比如应用本身的UI界面的语言,以及用户正在阅读的内容的语言

区别是  
封装为reaxel的库:整个应用共享这一个实例,比如reaxel-router  

封装为refaxel的库:实例化多个,之间互不影响,一般根据不同的需要,传入不同的参数.
